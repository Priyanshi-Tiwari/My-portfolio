What is HTML?

If you are a newbie who wants to learn web development or front-end web development, there are three things that are important to learn HTML, CSS and javascript but the first thing you need to understand is how HTML works. HTML stands for Hypertext Markup Language. Sound scary, right? But trust me it’s not. When it comes to learning and understanding html, it is the easiest of all. To understand what is Html, let’s first understand what is hypertext and markup? Hypertext: A hypertext is a link which when clicked takes you to another page. A number of interconnected hyperlinks is what creates the world wide web. Markup: html basically marks up your content with tags to tell the browser and the user how your file is structured. It tells where is paragraph using p tag or header using header tag, it also tells when the text is being bold or italized etc. To give your webpage the required formatting, HTML uses three main components tags, elements and attributes.
